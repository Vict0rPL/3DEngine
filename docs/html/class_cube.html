Detailed Description

Klasa reprezentująca sześcian w przestrzeni 3D.

Dziedziczy po klasie Object3D i implementuje metodę Draw(), umożliwiającą rysowanie sześcianu przy użyciu OpenGL.

◆ Draw()

void Cube::Draw ( )
overridevirtual

Rysuje sześcian w aktualnej scenie 3D.

Rysuje sześcian 3D z uwzględnieniem transformacji, teksturowania i zaznaczenia.

Funkcja nadpisuje metodę wirtualną z klasy Object3D. Wykorzystuje OpenGL i bibliotekę FreeGLUT do renderowania obiektu.

Funkcja:

  • Wczytuje macierz modelu obiektu.
  • Rysuje sześcian z włączonym teksturowaniem (jeśli aktywne).
  • Dodaje siatkę (wireframe), jeżeli obiekt jest zaznaczony.
  • Obsługuje rysowanie ścian z normalami i współrzędnymi UV.

Implements Object3D.
