Detailed Description

Reprezentuje teksturę 2D w OpenGL.

Ładuje teksturę z pliku przy użyciu biblioteki stb_image, udostępnia metody jej bindowania oraz usuwania.

void Texture2D::Bind ( ) const

Binguje tę teksturę jako GL_TEXTURE_2D na aktywnej jednostce teksturującej.

Wiąże (binduje) teksturę do aktualnego kontekstu OpenGL.

Po wywołaniu tej funkcji tekstura staje się aktywna i może być używana do renderowania.
